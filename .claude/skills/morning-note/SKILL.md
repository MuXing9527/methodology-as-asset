# 定位

晨报 Skill 做数据聚合，不做判断。
以下内容研究员手动补充：今日操作重点、个股具体观点。

# 步骤 1：宏观数据（公开来源）

- 隔夜美股三大指数涨跌（Wind）
- 美债 2Y/10Y 收益率变化（Wind）
- 美元指数 / 在岸人民币汇率
- 昨日 A 股收盘数据（沪深300/中证500/创业板指）

# 步骤 2：行业高频数据

- 按当日持仓行业，查询最近发布的高频数据
  （乘用车周销量、地产成交、猪价、铜价等——由 CLAUDE.local.md 配置）

# 步骤 3：公告扫描

- 持仓股昨晚公告列表（Wind 公告系统）
- 仅列标题 + 公告类型，不做解读（解读由人工完成）

# 步骤 4：日程提醒

- 今日财报披露（持仓相关）
- 今日重要数据发布（CPI / PPI / 社融等）

# 输出格式

仅 Markdown，不出 HTML。
模板化格式，每日一致，便于快速扫读。
