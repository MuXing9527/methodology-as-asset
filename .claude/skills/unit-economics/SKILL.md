# store_model（适用：零售 / 餐饮 / 连锁）

| 指标 | 含义 | 数据来源 |
|---|---|---|
| 单店收入（年）| 公司披露 / 调研估算 | 公告 / 行业报告 |
| 单店成本结构 | 租金 / 人力 / 原材料 | 公告附注 |
| 单店经营利润 | 推算 | 公式 |
| 开店成本 | 前期投入 | 公告 |
| 投资回收期 | 开店成本 / 年度经营利润 | 公式 |
| 系统性 ROI | 年利润 / 开店成本 | 公式 |

# per_customer（适用：SaaS / 金融服务 / 电商）

| 指标 | 含义 |
|---|---|
| CAC（获客成本）| 销售&营销支出 / 新增客户数 |
| LTV（客户终身价值）| ARPU × 毛利率 × 平均留存年限 |
| LTV/CAC 比率 | > 3 通常视为健康 |
| 回收期 | CAC / (ARPU × 毛利率) |
| NRR（净收入留存率）| 老客户收入变化 |

# per_unit（适用：制造 / 能源 / 大宗）

| 指标 | 含义 |
|---|---|
| 单位收入 | ASP（平均售价）|
| 单位成本 | 直接材料 + 直接人工 + 制造费用 |
| 单位毛利 | ASP - 单位成本 |
| 单位毛利率 | 单位毛利 / ASP |

# 重要约束

单位经济模型的核心假设（留存率 / ARPU / 单店收入）必须标注来源。
LLM 不自行估算这些数字，只能使用已有公开数据或研究员提供的数字。
