# 三层尽调清单结构

## 财务层

- [ ] 收入确认政策（与行业惯例是否一致）
- [ ] 应收账款周转天数趋势（近5年）
- [ ] 存货减值风险（近3年减值情况）
- [ ] 关联方交易规模和定价公允性
- [ ] 资本支出模式（扩张型 vs 维护型）
- [ ] 经营现金流 / 净利润质量比
- [ ] 有息负债结构（短债 / 长债 / 期限）

## 运营层

- [ ] 核心产品 / 服务的单位经济模型（单店 / 每客户 / 每单位）
- [ ] 产能利用率及扩产计划
- [ ] 供应链集中度（前5大供应商占比）
- [ ] 客户集中度（前5大客户占比）
- [ ] 人力资本：核心团队稳定性 / 离职率
- [ ] 研发投入强度和专利布局

## 竞争层

- [ ] 市场份额趋势（近5年）
- [ ] 主要竞争对手动向
- [ ] 定价权验证（涨价 / 降价历史）
- [ ] 进入壁垒评估（资质 / 规模 / 技术 / 客户关系）

# MNPI 标记规则

⚠ 以下项目在线下调研中可能涉及未公开信息（MNPI），**绝对不得输入任何 LLM**：
- 未公告的并购 / 重组计划
- 尚未披露的财务数据
- 内部预算 / 实际经营数据（非公开口径）

以上信息的判断和记录由研究员完全独立处理。

# 输出

以 Markdown 清单格式输出，每条附数据来源建议。
