# Skill: convertible-bond-analyzer

## 输入

- `bond_code`: 转债代码
- `mode`: `single_bond` / `weekly_pool` / `clause_watch` / `portfolio_review`

## 输出

- `output/convertible/{bond_code}_{mode}.md`
- `output/convertible/{bond_code}_{mode}.html`

## 步骤

### 步骤 1：数据清单

- [ ] 转债价格、平价、平价溢价率、纯债价值、纯债溢价率、YTM、隐含波动率
- [ ] 正股价格、行业、基本面、波动率、估值和催化
- [ ] 条款：下修、强赎、回售、剩余期限
- [ ] 供需：余额、成交、机构持仓、强赎退出压力

### 步骤 2：调 Subagent

`cn-convertible-bond-reader` 只抽取指标、条款和生命周期状态。

### 步骤 3：按平价区间分流

- 低平价：债底、信用、下修、YTM
- 中平价：性价比、正股弹性、估值、条款
- 高平价：跟涨能力、强赎风险、是否低效替代正股

### 步骤 4：四块价值拆解

1. 正股：上行逻辑是否成立，不确定性在哪里
2. 债底：纯债价值是否可信，信用风险是否会破坏保护
3. 估值：同平价、同行业、同评级、相近期限是否偏贵
4. 条款：下修、强赎、回售对收益分布的影响

### 步骤 5：输出结论

- 转债属性：债性/股债平衡/股性/退出阶段
- 核心机会：来自正股、估值修复、条款还是防守
- 核心风险：信用、估值压缩、强赎、流动性、正股下行
- 池子归类：低价防守池/中平价性价比池/高弹性池/下修潜力池/强赎风险池

## 检查清单

- [ ] 没有把正股观点直接等同为转债结论
- [ ] 债底可信度经过信用和流动性检查
- [ ] 同平价区间比较估值
- [ ] 高平价券检查强赎和低效替代风险
