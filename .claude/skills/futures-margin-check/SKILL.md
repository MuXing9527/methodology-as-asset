# 期货保证金的特殊时间要求

发现保证金使用率 > 80% 时，需要**当天**处理，不是第二天。
触发追保通知 → 24 小时内补足 → 否则强制平仓。

# 步骤 1：调 futures-margin-monitor Subagent

返回账户概览（使用率）+ 各合约明细（margin_to_force_close）

# 步骤 2：极端情景估算

对持仓最大合约，假设单日 3% 的不利波动（历史极端日约 2σ）：
额外保证金需求 = 合约价值 × 3% × 合约乘数 × 手数
当前可用资金 - 额外保证金需求 = 余量
