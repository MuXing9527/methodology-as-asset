# 重要约束

本 Skill 只做基本面条件筛选，不使用量化多因子、Barra 风险因子、机器学习预测。
不输出"买入"/"卖出"/"推荐"等结论性词汇。
输出是"满足条件的标的清单"，投资建议由研究员给出。

# 筛选条件（CLAUDE.local.md 可覆盖）

## 条件 A：基本面景气度

- 营收增速 YoY > X%（近2季均值）
- 毛利率 QoQ 改善 > Y%
- 经营现金流 / 净利润 > Z（排除利润虚高）

## 条件 B：估值合理性

- PE 不超过行业中位数 N 倍
- PB/ROE 相对估值处于历史中位以下

## 条件 C：机构关注度

- 近 30 天卖方研报覆盖 ≥ M 家（不过热、不被忽视）

## 条件 D：排除条件

- 近 12 月有重大违规 / 处罚公告
- 主要股东持续减持中
- 流动性不足（日均成交额 < P 万元）

# 步骤 1：数据查询

从 Wind 批量拉取 universe 股票的基本面指标。

# 步骤 2：条件筛选

逐条件筛选，保留同时满足 A+B+C 且无 D 的标的。

# 步骤 3：输出

列出满足条件的股票（ticker / 行业 / 满足条件列表），不附加任何投资判断。
研究员从此列表中自行决定是否深入研究。
