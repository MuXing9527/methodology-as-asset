# Skill: rates-curve-brief

## 输入

- `date`: 交易日或周度截止日
- `mode`: `daily` 或 `weekly`

## 输出

- `output/rates/{date}_rates_curve_brief.md`
- `output/rates/{date}_rates_curve_brief.html`

## 步骤

### 步骤 1：数据查询清单

- [ ] 收益率：国债 1Y/3Y/5Y/10Y/30Y，国开 10Y
- [ ] 曲线：10Y-1Y，30Y-10Y，10Y-DR007
- [ ] 资金：DR007、R007、OMO、MLF、缴税、跨季
- [ ] 供给：国债、地方债、专项债、政金债净融资
- [ ] 机构：银行、保险、理财、公募、农商行、外资净买卖

### 步骤 2：调 Subagent

`cn-rates-curve-reader` 只整理数据，不写方向判断。

### 步骤 3：五碗面判断

1. 基本面：增长、通胀、地产、出口、消费、投资
2. 政策面：货币、财政、地产、汇率约束
3. 资金面：DR007 是否偏离政策利率，R007-DR007 是否扩大
4. 供需面：政府债供给、配置盘、交易盘、赎回压力
5. 估值与曲线：期限利差、远期利率、期限溢价

### 步骤 4：曲线分段

- 短端：政策利率和资金面
- 中段：降息预期、交易盘久期、曲线下滑收益
- 长端：名义增长、期限溢价、保险/银行配置
- 超长端：保险负债久期、供给、波动率和流动性

### 步骤 5：输出结构

1. 一句话定调
2. 市场数据表
3. 五碗面复盘
4. 曲线分段解释
5. 和信用债/可转债的联动
6. 待验证事项和风险清单

## 检查清单

- [ ] 没有把牛陡/牛平/熊陡/熊平当分析起点
- [ ] 没有把远期利率当纯预测
- [ ] 没有输出买卖建议
- [ ] 缺失数据保留 `[VERIFY: ...]`
